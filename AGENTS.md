# General

## Package Manager

- Use `bun` for JS related dependencies and scripts

## Spell Check/Writing Improvement Rules

- Unless otherwise instructed, do not modify quotes even though there are
  writing issues.

## After Edits

- Run `bun format`
- Run `bun lint`

# Language Specific

## JavaScript

- Use kebab-case file names for JavaScript files in `prompts/skills`

## Typst

Always

- Run `typst compile <file>` to make sure the file compiles
- Run `typstyle --check <file>` to lint typst files

## Markdown

Beside the README file, Markdown files should be named using kebab cases.
